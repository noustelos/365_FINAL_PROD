**Σύνοψη Αλλαγών για την Ιστοσελίδα 365orthodoxy**

**1. Νέο Προϊόν: "365orthodoxy Widget"**
   - Παρουσιάστηκε ένα νέο προϊόν, το "365orthodoxy Widget".
   - Είναι ένα ψηφιακό Ορθόδοξο εορτολόγιο που παρέχει καθημερινό πνευματικό περιεχόμενο.
   - Μπορεί να ενσωματωθεί σε υπολογιστές, κινητά, tablet και ιστοσελίδες.

**2. Νέα Σελίδα Προορισμού (`index.html`)**
   - Δημιουργήθηκε μια νέα σελίδα προορισμού (landing page) για την προώθηση του widget.
   - Η σελίδα έχει τις ακόλουθες ενότητες:
     - **Κύρια Ενότητα (Hero Section):** Παρουσιάζει το widget με τίτλο, περιγραφή και κουμπί "Δείτε ζωντανά το widget".
     - **"Γιατί το δημιουργήσαμε;":** Εξηγεί το κίνητρο πίσω από τη δημιουργία του widget.
     - **"Βασικά Χαρακτηριστικά":** Απαριθμεί τις βασικές λειτουργίες:
       - Πρόσβαση σε έναν θησαυρό πνευματικών κειμένων.
       - Αναζήτηση Αγίων με το όνομά τους.
       - Λήψη ενός νέου τυχαίου αποσπάσματος για έμπνευση.
       - Αντιγραφή και κοινοποίηση περιεχομένου.
       - Εναλλαγή μεταξύ πρωτότυπου κειμένου και νεοελληνικής απόδοσης.
       - Σύνδεσμοι προς τις βιογραφίες των Αγίων στη Wikipedia.
     - **"Πνευματικές Πηγές":** Αναφέρει λεπτομερώς τις πηγές του περιεχομένου (Αγία Γραφή, Πατερικά Κείμενα, κ.λπ.).
     - **"Πώς θα το αποκτήσετε":** Μια φόρμα εγγραφής σε ενημερωτικό δελτίο για την ενημέρωση των χρηστών σχετικά με την κυκλοφορία.
     - **Υποσέλιδο (Footer):** Περιέχει πληροφορίες πνευματικών δικαιωμάτων.

**3. Διεθνοποίηση (i18n)**
   - Το περιεχόμενο της ιστοσελίδας διαχειρίζεται πλέον μέσω αρχείων JSON για εύκολη μετάφραση.
   - `translations/en.json` για τα Αγγλικά.
   - `translations/el.json` για τα Ελληνικά.
   - Ένα script (`script.js`) πιθανότατα χρησιμοποιείται για τον χειρισμό της εναλλαγής γλώσσας.

**4. Σχεδιασμός και Στυλ (`style.css`)**
   - Η ιστοσελίδα χρησιμοποιεί έναν νέο σχεδιασμό.
   - **Γραμματοσειρές:**
     - `GFS Didot` για επικεφαλίδες/κειμένου serif.
     - `Montserrat` για το υπόλοιπο κείμενο.
   - **Χρώματα:** Χρησιμοποιείται μια νέα παλέτα χρωμάτων, ορισμένη με μεταβλητές CSS (`--brand-bg`, `--white`, κ.λπ.).
   - **Φόντο:** Χρησιμοποιείται μια επαναλαμβανόμενη εικόνα φόντου (`assets/background_1.png`).
   - **Διάταξη:** Η διάταξη φαίνεται να είναι μοντέρνα και αποκριτική (responsive).

**5. Νέα Στοιχεία (Assets)**
   - Προστέθηκαν νέα αρχεία γραμματοσειρών στον κατάλογο `fonts/`.
   - Νέες εικόνες όπως `assets/background_1.png` και `assets/santorini_retro.jpeg`.

**Πώς να αναπαράγετε αυτές τις αλλαγές:**

1.  **Δημιουργήστε τη δομή HTML:** Δημιουργήστε ένα `index.html` με τις ενότητες που περιγράφονται παραπάνω. Χρησιμοποιήστε χαρακτηριστικά `data-translate` για το κείμενο που χρειάζεται να μεταφραστεί.
2.  **Δημιουργήστε τα αρχεία μετάφρασης JSON:** Δημιουργήστε τα αρχεία `translations/en.json` και `translations/el.json` με τα αντίστοιχα κλειδιά και τις μεταφρασμένες συμβολοσειρές.
3.  **Δημιουργήστε το CSS:** Δημιουργήστε ένα αρχείο `style.css` με τα στυλ για τις γραμματοσειρές, τα χρώματα, το φόντο και τη διάταξη.
4.  **Δημιουργήστε το JavaScript:** Δημιουργήστε ένα `script.js` για:
    -   Να φορτώνει τα αρχεία μετάφρασης JSON.
    -   Να υλοποιεί μια συνάρτηση για την εναλλαγή της γλώσσας.
    -   Να γεμίζει το HTML με το μεταφρασμένο κείμενο με βάση την επιλεγμένη γλώσσα.
    -   Να χειρίζεται τη φόρμα εγγραφής στο ενημερωτικό δελτίο.
5.  **Προσθέστε τα Στοιχεία (Assets):** Προσθέστε τις νέες γραμματοσειρές και εικόνες στο έργο.
